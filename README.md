# ttyrec2gif

![ttyrec2gif](https://raw.githubusercontent.com/mattn/ttyrec2gif/master/screenshot.gif)

create animated gif from ttyrecord.

## Usage

```console
$ ttyrec2gif ttyrecord
```

## Requirements

* libgd
* libvterm

## Installation

```console
$ make
```
